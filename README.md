# Batman Hackathon

Batman: A Phaser 3 Demo 

Begin the experience as the Dark Knight of Gotham on our Batman platformer.

Buttons:

    Move Left- Left arrow
    
    Move Right - Right arrow
    
    Jump - Up arrow
    
    Punch - Spacebar

Prerequisites
`Node`
`JavaScript`

Built With
`Express` - The web framework used
